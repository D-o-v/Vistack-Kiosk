How it works:

  1. Email Link: User clicks reset password link from email
  2. Redirect Page: Link goes to /reset-password.html first
  3. Parameter Validation: Page validates token and email parameters
  4. Main App: Redirects to main app with reset=true parameter
  5. Reset Form: Main app shows reset password form
  6. API Call: Form submits to backend API
  7. Success: User is redirected to login page

Sample Email Link Format:

http://localhost:5173/reset-password.html?token=RESET_TOKEN&email=USER_EMAIL

Backend Should Generate:

http://checkin.vistacks.com/reset-password.html?token=RESET_TOKEN&email=USER_EMAIL
